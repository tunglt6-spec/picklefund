- **[✨ VDS-01 — Visual Design System v2.1](design/VDS-01-visual-design-system-v2.1.md)** — **Visual Constitution** bổ trợ UDP-01: visual quality bar (Linear/Stripe/Vercel/Notion), 10 visual principles, layout rhythm, KPI/table/mobile/AI/finance/data-viz standard, Visual Acceptance Checklist + Visual Score Gate, Taste Skill Bootstrap Rule. **✅ Accepted / Codex PASS**; áp dụng từ UIP-05/UI-05 trở đi.
