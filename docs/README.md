# PickleFund — Tài liệu kỹ thuật

Chỉ mục tài liệu của dự án PickleFund.

## Bộ tài liệu

- **[⚖️ GOV-01 — Project Governance Baseline v2.1](governance/GOV-01-project-governance-baseline-v2.1.md)** — **Single Source of Truth** cho toàn bộ Project Governance (17 rules, principles, workflow, lifecycle, pre-audit checklist). Mọi tài liệu khác chỉ **tham chiếu** GOV-01.
- **[🎨 UDP-01 — Unified Product Design Constitution](design/UDP-01-unified-product-design-constitution.md)** — **Design Source of Truth** (tokens, accent theo module, layout, card/table/filter, responsive, Feature Parity, shared component library). **Foundation ✅ Codex PASS / CLOSED.** UI mới chỉ tham chiếu UDP-01.
- **[🧊 DESIGN-01 — Design Foundation Freeze Declaration](design/DESIGN-01-design-foundation-freeze-declaration.md)** — tuyên bố đóng băng Design Tokens + Shared Components; mọi thay đổi Foundation phải qua Design Increment mới. UI-02 Dashboard 4.0 = increment tiếp theo (chưa triển khai).
- **[📌 PROJECT STATUS](PROJECT_STATUS.md)** — trang điều hành: trạng thái dự án, kiến trúc, nguyên tắc, quy trình, roadmap. Bắt đầu tại đây.
- **[Technical Baseline v2.0](technical-baseline-v2.0/README.md)** — điểm chốt kỹ thuật sau Sprint 2 / Epic 2.4 (Codex Second Final Re-Audit PASS). Bắt đầu đọc tại đây để nắm trạng thái và các bất biến kiến trúc hiện tại.
- **[Sprint 4 — ADR-01 Execution Engine Architecture](sprint4-execution-engine/ADR-01-execution-engine-architecture.md)** — kiến trúc Execution Engine (Codex PASS; chưa triển khai code).
- **[Sprint 4 — ADR-02 Execution Governance Model](sprint4-execution-engine/ADR-02-execution-governance-model.md)** — mô hình quản trị Execution: maturity level, allowlist/denylist, approval, readiness (Codex PASS; chưa triển khai code).
- **[Sprint 4 — ADP-01 Decision to Proceed](sprint4-execution-engine/ADP-01-decision-to-proceed.md)** — quyết định quản trị: cho phép mở **duy nhất Epic 4.1** (Execution Ticket Framework, không execute); Execution Readiness vẫn NOT READY.
- **[Sprint 4 — ADR-03 Execution State Strategy](sprint4-execution-engine/ADR-03-execution-state-strategy.md)** — chiến lược state machine cho Epic 4.2 (Pure State Machine; Codex PASS / Accepted; chưa triển khai code, Epic 4.2 chưa mở).
- [V2.1 AI Brain](V2.1_AI_BRAIN/INDEX.md) — đặc tả & báo cáo Sprint 1/2 của lớp AI.
- [Commercial Release](commercial-release/RELEASE_HANDBOOK.md) — sổ tay phát hành, hướng dẫn admin/user/API/cài đặt.
