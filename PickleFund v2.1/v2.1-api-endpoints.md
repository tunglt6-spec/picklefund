## 📋 AI ACTION ENDPOINTS

### 1. Create Action (AI → Action Center)
**POST** `/actions`

**Called by:** Maika AI, Lisa AI, Hermes
**Description:** AI tạo action recommendation

**Request:**
```json
{
  "clubId": "club-123",
  "proposedBy": "MAIKA",
  "actionType": "SEND_REMINDER",
  "targetId": "member-xyz",
  "title": "Nhắc đóng quỹ",
  "description": "Thành viên chưa đóng quỹ kỳ 1",
  "payload": {
    "reminderType": "payment",
    "dueDate": "2024-01-20",
    "amount": 500000
  },
  "riskLevel": "LOW",
  "confidenceScore": 0.95,
  "estimatedImpact": "Increase collection by 20%",
  "successCriteria": "Payment received within 7 days"
}
```

**Response:**
```json
{
  "id": "act-12345",
  "status": "PENDING",
  "requiresApproval": false,
  "createdAt": "2024-01-15T08:00:00Z"
}
```

**Status Codes:**
- `201 Created` - Success
- `400 Bad Request` - Invalid payload
- `401 Unauthorized` - Invalid AI token
- `404 Not Found` - Club not found

## ✅ APPROVAL ENDPOINTS

### 4. Approve Action
**POST** `/actions/:actionId/approve`

**Caller:** Admin / Manager (via Dashboard or Telegram)

**Request:**
```json
{
  "approverId": "user-admin",
  "comment": "Approved - send reminder now"
}
```

**Response:**
```json
{
  "id": "act-12345",
  "status": "APPROVED",
  "approvedAt": "2024-01-15T08:15:00Z",
  "nextStep": "Execute within 2 minutes"
}
```

## 🎯 TELEGRAM BOT ENDPOINTS (MÍT ĐẶC)

### 14. Telegram Webhook
**POST** `/telegram/webhook`

**Called by:** Telegram servers (incoming messages)

**Payload:** Standard Telegram update

**Responses:**
```
/pending          → List pending actions
/approve <id>     → Approve action (with inline keyboard)
/reject <id>      → Reject action
/actions          → List all actions
/balance          → Check club balance
/debt             → List debts
```

---

## 🔐 AUTHENTICATION

All endpoints require:
```
Authorization: Bearer {jwt_token}
```

**Token must include:**
```json
{
  "sub": "user_id | ai_name",
  "role": "ADMIN | MANAGER | MEMBER | AI",
  "clubId": "club-123"
}
```

---

## 📊 RATE LIMITING

- Admin endpoints: 100 requests/minute
- AI endpoints: 50 requests/minute
- User endpoints: 20 requests/minute

---

## ⚠️ ERROR RESPONSES

All errors follow standard format:
```json
{
  "error": "Error code",
  "message": "Human-readable message",
  "details": { ... }
}
```

**Common error codes:**
- `INVALID_PAYLOAD` - 400
- `UNAUTHORIZED` - 401
- `FORBIDDEN` - 403
- `NOT_FOUND` - 404
- `CONFLICT` - 409
- `RATE_LIMITED` - 429
- `INTERNAL_ERROR` - 500
