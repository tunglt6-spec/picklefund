# Assets — PickleFund V2.0 RC1

> Thư mục này chứa tài nguyên branding và UI assets của PickleFund V2.0 RC1.

---

## Danh sách assets dự kiến

| File | Mô tả |
|---|---|
| `logo.svg` | Logo PickleFund (SVG vector) |
| `logo-dark.svg` | Logo nền tối (dark theme) |
| `logo-light.svg` | Logo nền sáng (light theme) |
| `favicon.ico` | Favicon cho trình duyệt |
| `app-icon-512.png` | App icon 512×512 (PWA) |
| `og-image.png` | Open Graph image cho social sharing |
